# 开发进度记录

## 当前状态概览

项目已经完成可运行的 Web/PWA 本地版本，并能构建 Android debug APK。主线功能和最近一轮 UI 细节基本稳定，后续主要进入维护、体验微调、功能补强阶段。

## 已完成功能

### 播放器

- 首页播放器支持播放、暂停、上一首、下一首、播放模式、进度条、音量、歌词预览。
- 播放详情页支持大歌词、歌词定位播放、喜欢状态、播放队列抽屉。
- 播放进度在重新进入软件时不再自动恢复到上次秒数，默认从 `0` 开始。
- 当前播放行按下/点击时有短暂半透明反馈，不做持续选中态。

### 歌词

- 首页歌词和播放详情页歌词均使用整体滑动体验。
- 歌词加载中状态应居中展示。
- 歌词选中后再点击播放按钮才触发快进，未选中时不应误触发。
- 去掉歌词播放按钮点击后的系统蓝色焦点框。

### 网易云能力

- 支持登录状态、用户资料、歌单列表、歌单详情、喜欢歌曲。
- 支持搜索歌曲、播放搜索结果、收藏/取消收藏。
- 支持歌手详情页，包括歌手信息、热门歌曲、专辑列表。
- 支持从歌手详情页点击专辑进入独立专辑详情页。
- 专辑详情页当前采用列表模式，不使用九宫格。
- 喜欢歌曲爱心图标和逻辑应与首页播放器保持一致。

### UI 布局

- 顶部固定栏高度已做过全局收窄，目标是刚好容纳按钮/输入框并留少量冗余。
- 全局返回键应收在统一位置，以歌手详情页位置为准。
- 搜索框只在点击搜索按钮后展开，再次点击或空输入时点击外部自动关闭。
- 搜索框宽度需要避免压到搜索按钮和设置按钮。
- 歌单详情页、搜索结果页、歌手详情页、专辑详情页顶部间距应保持一致。
- 歌曲列表整体尺寸和歌手详情里的列表风格保持同步。

### Android

- Capacitor Android 工程已存在。
- `android-node/` 用于打包本地网易云 API 能力。
- 最近一次 debug APK 构建成功，输出路径为：

```text
android/app/build/outputs/apk/debug/app-debug.apk
```
